-> A linguagem possui o paradigma de orientacao a objetos
-> O objeto 'assert' é utilizado para testar condicoes, com o metodo 'equal'
   ex: assert.equal(7+1, 8)
-> Operadores: '&&' - AND
	       '||' - OR
               '<, <=, ===, !==' - COMPARADORES

-> As variaveis nao possuem tipo fixo
-> Declaracao:
	 let x; //declara a variavel x
	 let y = 3 * 5; //como em C permite inicial a variavel na declaracao
	 const Z = 8; //declara uma constante

-> Em muitos aspectos é igual a C, por exemplo if e operador ternario 

-> os tipos primitivos (boolean, string, number) são imutaveis, ou seja, funcionam 
basicamente como uma constante, quando declaramos, por exemplo
  let a = 10;
estamos copiando 10 para a variavel a, ou seja, sempre passamos por valor
-> os objetos sao mutaveis
